**결론**

Logistic Regression은 선형 모델이라서 입력 정보가 많을수록 더 다양한 관계를 학습할 수 있어서 전체 특성을 썼을 때 성능이 더 좋게 나왔고, 반대로 Decision Tree나 Random Forest 같은 트리 기반 모델은 특성이 많아지면 복잡해지면서 과적합이나 노이즈 영향이 커져서 오히려 성능이 떨어질 수 있어서 corr처럼 핵심 특성만 사용했을 때 더 좋은 결과가 나왔다. 그래서 단순히 feature를 많이 쓰는 것보다 모델에 맞게 적절하게 선택하는 게 더 중요하다고 생각된다.
